# Contributing

Thank you for taking the time to contribute to this project!

## Code of Conduct

This project adheres to the Contributor Covenant [code of conduct](CODE_OF_CONDUCT.md).
By participating, you are expected to uphold this code.
Please report unacceptable behavior to opensource@kensho.com.

## Style Guide

- **Python** code follows the [Google Python Style Guide](https://google.github.io/styleguide/pyguide.html).
- **Rust** code follows the standard [Rust style guidelines](https://doc.rust-lang.org/nightly/style-guide/);
  format with `cargo fmt` and lint with `cargo clippy` before submitting.
